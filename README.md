# FM Csound Synthesis

Codice rielaborato dal Manuale della Musica Elettronica - Pousseur
- fm02 modello base
- fm03 freq base con intorno
- fm04 im e ip con intorno
- fm05 index intorno
- fm06 intorno tutto
